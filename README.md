## Minijuego Guille: Esquivar las pelotas

El proyecto consiste en un juego sencillo en el que el personaje tiene que esquivar las pelotas que van cayendo del cielo de manera aleatoria. El juego se inicia con 3 vidas y cada vez que una pelota toca al personaje en cualquier parte del cuerpo, pierde una vida. En el momento que le tocan 3 veces, el juego se acaba y aparece la pantalla de Game Over. 
Como posibles mejoras, se podrían introducir niveles, donde cada vez que alcance un número determinado de puntos, cambie de pantalla y las pelotas caigan más rápido para dificultar el juego. También se podría meter objetos que cayeran y dieran puntos al personaje.

**Instrucciones del juego:**
1. Presiona la <ins>bandera verde</ins>.
2. Mueve al jugador con las flechas _izquierda/derecha_.
3. Evita que las pelotas te golpeen.
4. Ganas puntos por cada pelota que cae sin tocarte.
5. Cada vez que te toque una pelota, pierdes una vida.
6. Cuando te golpean <ins>3 veces</ins>, el juego termina y aparece la pantalla **GAME OVER**.
